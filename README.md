# 🎮 Short Story Game

A beginner-friendly Python mini game that creates a fun story using user inputs.

## 🧠 Concepts Used

* `input()`
* Variables
* Strings
* `print()`
* Escape sequence (`\n`)

## 🚀 How It Works

The user enters:

* Name
* Favourite place
* Favourite food

Python then generates a funny short story using those inputs.
